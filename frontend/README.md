# Hextech Ledger

A fan-made competitive and learning hub for the TCG Riftbound. Players can search for gameplay videos by champion or event, from official or high quality sources. Associated decklists and rulings (when available), will accompany these videos to help explain/expose new tecniques and playstyles as well.

Uses React, Next.js, Typescript and Tailwind.

## Getting Started

> Uses:
>
> - Node v24.12.0
> - yarn

In your terminal:

1.  `yarn`
2.  `yarn dev`

The app should now be served on [localhost:3000](http://localhost:3000/)
